# Contributing to mini_ml

Thank you for your interest in contributing!

## How to Contribute
1. **Fork** the repository and create a new branch from `main`.
2. **Write or improve code** in the `src/` folder.
3. **Add tests** for your code in the `tests/` folder. Follow the style of existing test files.
4. **Run all tests**:
   ```bash
   pytest
   ```
   Ensure all tests pass. Take a screenshot of your terminal showing all tests passing and include it in your pull request.
5. **Use semantic commit messages** ([guide](https://www.conventionalcommits.org/en/v1.0.0/)).
6. **Open a pull request** with a clear description of your changes.

## Code Style
- Keep code simple and readable.
- Add docstrings and comments where helpful.
- Follow PEP8 where possible.

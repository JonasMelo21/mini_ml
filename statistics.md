# Statistics

Basic statistics utilities for ML.

## Theory & Math
Statistics help summarize and understand data. This module provides mean, variance, etc.

## Usage
```python
from src.statistics import mean, variance

X = [1, 2, 3, 4, 5]
print(mean(X))
print(variance(X))
```

## Functions
- `mean(data)`
- `variance(data)`
- `std_deviation(data)`

See the source code for more details.
